# Spring Boot 博客系统
基于 Spring Boot 3 + MyBatis-Plus 实现的前后端分离博客后端系统，支持用户认证、文章管理、权限控制等核心功能。

## 🚀 技术栈
| 技术 | 版本 | 说明 |
|------|------|------|
| Spring Boot | 3.x | 后端核心框架 |
| MyBatis-Plus | 3.5.x | 持久层框架 |
| MySQL | 8.x | 关系型数据库 |
| JWT | - | 用户登录鉴权 |
| Knife4j | 4.x | 接口文档生成 |
| Lombok | - | 简化Java代码 |

## ✨ 核心功能
### 用户模块
- 用户注册/登录
- JWT Token 身份认证
- 登录状态拦截与权限控制

### 文章模块
- 文章发布（仅登录用户可操作）
- 文章列表分页查询
- 文章修改/删除（仅作者本人可操作）
- 数据权限控制（只能操作自己的文章）
